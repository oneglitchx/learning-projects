# Python Resources

This directory contains resources for learning and experimenting with Python. It is divided into two main subdirectories, each with a different focus.

| Directory | Short Description |
| --- | --- |
| [learn-python](#learn-python) | A structured, test-driven playground and cheatsheet for learning Python from the ground up. |
| [Python](#python) | A large collection of individual Python scripts for a wide variety of tasks and purposes. |

---

## learn-python

The `learn-python` directory provides a comprehensive and structured approach to learning Python. It is organized by topic, with each topic containing code examples, explanations, and links to further readings.

### Key Features:

- **Playground:** You can experiment with the code, change it, and test it to see how it works.
- **Cheatsheet:** The code examples serve as a handy reference for Python syntax and standard library usage.
- **Test-Driven:** The repository uses `pytest` for testing, allowing you to verify your understanding and the correctness of your code.
- **Linting:** It encourages good coding practices by using `pylint` and `flake8` to ensure your code adheres to the PEP 8 style guide.

This directory is ideal for those who want to learn Python in a structured, hands-on manner.

## Python

The `Python` directory is a large collection of individual Python scripts that have been contributed by various authors. It serves as a repository of examples and small, ready-to-use programs for a wide range of tasks.

### Content:

- **Diverse Scripts:** You can find scripts for everything from web scraping and automation to games and GUI applications.
- **Educational Examples:** Many scripts are simple enough for beginners to understand and learn from.
- **Practical Utilities:** Some scripts are designed to solve common problems and can be used as-is.

This directory is a great resource for finding code snippets, learning how to implement specific functionalities, or just exploring the possibilities of Python.
